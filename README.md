# Pytorch-cifar100

practice on cifar100 using pytorch

## Requirements

This is my experiment eviroument
- python3.6
- torch1.13.1+cu117
- tensorboard 2.13.0
- vit-pytorch 1.7.0

## Usage

### 1. enter directory
```bash
$ cd pytorch-cifar100
```



### 3. train the model
Go to `main.py` and check the parameters at the beginning of the file, then run:
```
python main.py
```
